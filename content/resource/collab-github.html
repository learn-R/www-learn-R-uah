Objetivo del material

Este material busca ser un apoyo introductorio a la lógica de Git y GitHub, profundizando en los contenidos de la primera sesión del curso para que puedas aprovechar las herramientas que tiene GitHub para trabajar colaborativamente. La presentación se basa en Git for Humans, un conjunto de diapositivas creadas por Alice Bartlett en 2016.
